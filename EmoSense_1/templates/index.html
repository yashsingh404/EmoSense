⚡ Quick Setup Guide

  1. Download the FER2013 dataset from Kaggle → place fer2013.csv in the data/ folder
  2. Install dependencies: pip install -r requirements.txt
  3. Train the model: python train_model.py — takes ~30–60 min on GPU
  4. Start the server: python app.py — then open http://localhost:5000
  5. Click Start Camera below — allow webcam access and see your emotions live!
